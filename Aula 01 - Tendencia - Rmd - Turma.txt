## Estimando a tendência

**Exemplo do Morettin**

Para fazer o vetor da tendência determinística (linear) iremos utilizar o comando `seq(from,to,length)`, contudo `seq(from,to):` também gerará uma sequência. `by=`específica o incremento, `length=` específica o tamanho desejado da sequência e `seq(along=x):` gera uma sequência 1,2,...até x. ideal para o loop for

```{r vetor tendencia}
t <- seq(1,24,length=24)
t 
```
Note que, conforme o exemplo do livro Morettin & Toloi, precisamos utilizar os dados mensais de consumo de energia entre 1977:jan e 1978:dez. O comando `n <- length(t)` pode  criar um item que represente o tamanho da amostra n.

```{r n}
n <- length(t) # criando o n
n
```
